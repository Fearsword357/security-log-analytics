# Security Log Analytics

A Python-based tool that simulates and analyzes authentication logs to detect potential brute-force attacks.

## Features

- Generates realistic authentication logs
- Detects failed login attempts by IP
- Identifies potential brute-force attacks
- Visualizes attack patterns over time using matplotlib

## Example Output

Failed Login Attempts by IP:
192.168.1.50 -> 12  
203.0.113.77 -> 14  

Possible Brute Force Alerts:
ALERT: 192.168.1.50 has 12 failed login attempts  
ALERT: 203.0.113.77 has 14 failed login attempts  

## Installation

```bash
pip install -r requirements.txt
